OL Job Alert SaaS

A multi-tenant SaaS platform for OnlineJobs.ph job alerts with Telegram bot integration. Users bring their own Telegram bot, subscribe to keywords via a web dashboard, and receive real-time alerts when jobs match.
Built with Django 4.2, PostgreSQL LISTEN/NOTIFY, and Celery for real-time, event-driven job matching.

View on GitHub

Project Overview

Introduction

     OLJAlerts SaaS transforms the concept of job alerts into a multi-tenant web application where each user brings their own Telegram bot. Users register via web or Google OAuth, connect their bot token, manage keyword subscriptions through a modern dashboard, and receive real-time alerts powered by PostgreSQL's LISTEN/NOTIFY mechanism. Built with Django 4.2, Celery, and Redis, the platform offers a freemium model with robust security and modern UI.

Challenges

⏺ Multi-Tenant Architecture: Building a system where each user has their own Telegram bot while maintaining data isolation and security.
⏺ Real-Time Processing: Implementing instant job matching and alerts without polling overhead or delays.
⏺ Bot Security: Encrypting bot tokens at rest and validating connectivity without exposing credentials.
⏺ Subscription Management: Implementing tiered keyword limits and user quota enforcement.
⏺ User-Friendly Bot Setup: Simplifying bot connection without requiring webhooks or public URLs.

Solution

     The solution uses PostgreSQL LISTEN/NOTIFY triggers for real-time event-driven architecture. When a new job is scraped from OnlineJobs.ph, a trigger fires a NOTIFY event that a dedicated Celery worker listens for. The worker processes jobs, matches them against user keywords using word-boundary regex, and delivers alerts via Telegram API. Bot setup is simplified using @getidsbot for Chat ID (no webhooks needed), and all bot tokens are encrypted with cryptography.fernet. The entire system runs in Docker Compose with data persistence, featuring a modern Tailwind CSS UI.

Key Features

✔ Multi-Tenant Bot System: Each user connects their own Telegram bot with encrypted token storage.
✔ Real-Time Alerts: PostgreSQL LISTEN/NOTIFY triggers for instant job processing - no polling.
✔ Web-Based Keyword Management: Full CRUD (Add, List, Delete, Toggle) with validation and tier limits.
✔ Subscription Tiers: Free tier (3 keywords) and Contributor tier (unlimited) with charity donations.
✔ Simplified Bot Setup: Uses @getidsbot for Chat ID - no webhooks or public URL required.
✔ Modern UI: Tailwind CSS sidebar layout with responsive design and job statistics.
✔ Automated Scraping: Built-in OnlineJobs.ph scraper with rate limiting and duplicate detection.
✔ Alert History: Track sent alerts, prevent duplicates, automatic cleanup after 2 weeks.
